# Premium Wedding Invitation Website

This is a static animated wedding invitation website inspired by a cream, rose, and gold Sri Lankan wedding invitation style.

## How to run

Open `index.html` directly in a browser, or use VS Code Live Server.

## How to customize

Click the **Customize** button on the top-right side. You can change:

- Bride name
- Groom name
- Parents' names
- Guest name
- Wedding date and time
- Poruwa time
- Venue
- Location
- Google Maps link
- Final wish text

After editing, click **Apply Changes**.

## Client URL

Click **Copy Client URL** after entering the client details. Send that copied URL to the client. The URL contains the changed invitation details.

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Go to Vercel and import the repository.
3. Framework preset: Other / Static.
4. Deploy.

## Files

- `index.html` - main page
- `css/style.css` - full premium design and animations
- `js/app.js` - customizer, URL sharing, petals animation, scroll reveal
